ASSIGNMENT
Title
Conceptual IoT Monitoring System Design
1. Background / Problem Statement
IoT devices generate continuous sensor data from factories, smart devices, and monitoring systems. Organizations face challenges such as:
Monitoring devices in real time
Storing large volumes of sensor data
Identifying abnormal readings
Displaying live monitoring dashboards
Managing structured and semi-structured data
The organization wants to design a simple IoT Monitoring System Architecture using database, streaming, and analytics concepts.
2. Overall Objective
Design a beginner-friendly IoT monitoring solution that:
Collects sensor data
Stores sensor readings
Displays monitoring dashboards
Detects abnormal sensor values
Demonstrates data flow understanding using SQL/NoSQL concepts
3. Learning Outcomes
By completing this assignment, students will be able to:
Understand IoT architecture basics
Design simple data flow diagrams
Understand SQL vs NoSQL usage
Create basic monitoring database designs
Understand real-time analytics concepts
Build simple reporting dashboards
Explain end-to-end IoT monitoring workflows
